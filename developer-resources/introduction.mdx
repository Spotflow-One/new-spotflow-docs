## API Keys

Use the appropriate API keys :

- Test: `sk_test_fXXXXedhXXXXXXXXXXXXX`
- Live: `sk_live_fXXXXedhXXXXXXXXXXXXX`

Learn more about [Test vs Live Keys](/miscellaneous/test-mode-vs-live-mode).

## Integration Steps

Follow these steps to integrate Spotflow:

<Steps>
  1. [Set up your account](https://app.spotflow.co/) and get your API keys
  2. Choose your integration method:
     - [Use our SDKs](/developer-resources/spotflow-sdks)
     - [Direct API integration](/developer-resources/integration-guide)
  3. [Set up webhooks](https://app.spotflow.co/) for real-time updates
  4. [Test your integration](/miscellaneous/testing-process)
  5. Go live!
</Steps>
